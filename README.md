# Opensea Blockchain Clone App

**Opensea Blockchain Clone App**은 Opensea의 기능들을 클론 코딩한 프로젝트 입니다.  
**MetaMask**를 통해 결제가 가능하며 **thirdweb**에서 상품 관리가 가능합니다.  
**Sanity**로 구현된 관리자 페이지에서 User 관리가 가능합니다.
